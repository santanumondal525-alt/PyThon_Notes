# 📊 1️⃣ Line Chart

## 📖 What is a Line Chart?

A Line Chart shows how data changes over time.

It connects data points using lines.

---

## ✅ Best Used For

* Time series analysis
* Trends over time
* Growth or decline analysis

---

## 📊 Example

Monthly sales, stock prices, temperature changes, Google Trends.

* Trend
* Volume

* Showing contribution over time

* Minimum value
* Maximum value
* Median
* Quartiles
* Outliers
